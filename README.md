# Catching up with JUnit at JCON Europe 2025

Code for the exercises of the [JUnit workshop](https://schedule.jcon.one/session/837895) at JCON Europe 2025.

## Requirements

- JDK 21

## Building

The project can be built with Gradle or Maven.

### Gradle

```shell
./gradlew build
```

### Maven

```shell
./mvnw verify
```
